Particionamento de Conjuntos
============================

Isto é um trabalho acadêmico (Ciência da Computação) implementado em 2007, que aplica Simulaed Annealing e Algoritmos Genéticos para realizar particionamento de conjuntos.

O programa foi escrito em C#, na época foi utilizado o Visual Studio 2005, agora já está convertido para VS2015, fique livre para melhorá-lo.
